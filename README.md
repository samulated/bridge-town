# Bridge Town
The initial idea for Bridge Town is this: what if those quaint little fantasy villages you see in games like RuneScape had a fully simulated back-end with villager needs, interdependant crafts-people, and an economy. What if that simulation also existed in the context of a larger world.

Simply: robust, interconnected systems with a cozy low-poly medieval fantasy overlay on top.
